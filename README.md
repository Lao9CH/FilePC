# WebOS 教学系统服务器

一个基于 Web 的文件管理系统,用于局域网环境下的教学文件共享和管理。

## 功能特性

- 📁 **文件浏览** - 浏览服务器上的文件和文件夹
- ⬆️ **文件上传** - 支持拖拽上传和点击上传
- ⬇️ **文件下载** - 快速下载文件
- 👁️ **文件预览** - 支持图片、视频、音频、PDF、文本文件预览
- 📂 **文件夹管理** - 创建、删除、重命名文件夹
- ✏️ **文件管理** - 删除、重命名文件
- 🎠 **图片轮播** - 图片文件夹支持轮播模式
- ⚡ **虚拟滚动** - 大量文件时自动优化性能

## 快速开始

### 方法一:使用启动脚本(推荐)
1. 右键点击 `启动系统.bat`
2. 选择"以管理员身份运行"
3. 在浏览器中访问显示的地址

### 方法二:直接运行
1. 右键点击 `web-os-server.exe`
2. 选择"以管理员身份运行"
3. 在浏览器中访问 `http://localhost`

## 访问地址

- **本机访问**: http://localhost
- **局域网访问**: http://[服务器IP地址]

启动后会在命令窗口显示具体的访问地址。

## 系统要求

- Windows 7 及以上版本
- 需要管理员权限(使用80端口)
- 现代浏览器(Chrome、Edge、Firefox等)

## 注意事项

1. 首次运行需要以管理员身份运行
2. 默认管理 `D:\CDZ` 目录下的文件
3. 关闭命令窗口即可停止服务器
4. 请勿在服务器运行时删除或移动程序文件

## 常见问题

**Q: 双击后窗口闪退?**  
A: 请右键选择"以管理员身份运行"

**Q: 提示端口被占用?**  
A: 关闭其他使用80端口的程序(如IIS、Apache等)

**Q: 局域网无法访问?**  
A: 检查防火墙设置,确保允许80端口访问
