================================================================================
                    WebOS 教学系统服务器 - 使用说明
================================================================================

【系统简介】
这是一个基于 Web 的文件管理系统,可以通过浏览器访问和管理本地文件。
适用于局域网环境下的教学文件共享和管理。

【系统要求】
- 操作系统: Windows 7 及以上版本
- 需要管理员权限运行(使用80端口)
- 浏览器: Chrome、Edge、Firefox 等现代浏览器

【安装步骤】
1. 将整个文件夹解压到任意位置
2. 确保 D:\CDZ 目录存在(系统会自动创建)
3. 右键点击 "web-os-server.exe",选择"以管理员身份运行"

【启动方法】
方法一(推荐):
  - 右键点击 "web-os-server.exe"
  - 选择 "以管理员身份运行"
  - 等待命令窗口显示服务器地址

方法二:
  - 双击运行 "web-os-server.exe"
  - 如果提示权限错误,请使用方法一

【访问方法】
启动成功后,命令窗口会显示访问地址:
  - 本机访问: http://localhost
  - 局域网访问: http://[显示的IP地址]

在浏览器中输入上述地址即可访问系统。

【功能说明】
✓ 文件浏览 - 查看 D:\CDZ 目录下的所有文件和文件夹
✓ 文件上传 - 支持拖拽上传和点击上传
✓ 文件下载 - 点击文件即可下载
✓ 文件预览 - 支持图片、视频、音频、PDF、文本文件预览
✓ 文件夹管理 - 创建、删除、重命名文件夹
✓ 文件管理 - 删除、重命名文件
✓ 图片轮播 - 图片文件夹支持轮播模式查看
✓ 虚拟滚动 - 大量文件时自动优化性能

【快捷方式】
桌面图标:
  - "我的文档" - 快速访问 D:\CDZ\我的文档
  - "练习" - 快速访问 D:\CDZ\练习

【注意事项】
1. 首次运行需要以管理员身份运行(因为使用80端口)
2. 如果80端口被占用,请关闭其他使用80端口的程序
3. 所有文件操作都在 D:\CDZ 目录下进行
4. 请勿在服务器运行时删除或移动程序文件
5. 关闭命令窗口即可停止服务器

【故障排除】
问题1: 双击后窗口闪退
解决: 右键选择"以管理员身份运行"

问题2: 提示端口被占用
解决: 关闭其他使用80端口的程序(如IIS、Apache等)

问题3: 浏览器无法访问
解决: 
  - 检查服务器是否正常运行
  - 检查防火墙设置
  - 确认访问地址正确

问题4: 局域网其他设备无法访问
解决:
  - 检查防火墙是否允许80端口
  - 确认使用正确的局域网IP地址
  - 确保设备在同一局域网内
